# ⏰ Smart Voice Alarm Pro
PWA con estética Neón, segundero de alta precisión y detección automática de territorio.

## 🚀 Instalación
1. Sube los archivos a GitHub.
2. Activa GitHub Pages.
3. Añade a la pantalla de inicio en tu móvil.
4. **Toca la pantalla una vez** para habilitar el audio y el WakeLock.
